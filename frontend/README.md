
# Lenspost Admin Dashboard

This is the official repository for Lenspost Admin



## For Local Development

 Clone the Repo and Install all the dependencies

```bash
    1. cd backend 
```
```bash
    2. yarn && yarn dev
```

Run the backend service and Start the Frontend server

```bash
    1. yarn && yarn dev
```
